# ESP32_AirQuality
An open-source, ultra-low-power environmental dashboard for monitoring PM2.5, CO2 equivalents, and workspace VOCs.

#✨Key Features

Multi-Sensor Integration:

PMS5003: High-precision laser dust sensor for PM1.0, PM2.5, and PM10.

ENS160: Advanced digital gas sensor for TVOCs and equivalent CO2 (eCO2).

BME280: Environmental sensor for Temperature, Humidity, and Barometric Pressure.

Low-Power E-Ink Display: Provides a clear, paper-like reading experience that consumes power only when refreshing the screen.

Over-The-Air (OTA) Updates: Update your firmware wirelessly via Wi-Fi without needing to plug in a USB cable.

Open-Source Hardware: Fully 3D printable enclosure designed for easy assembly and optimal sensor airflow.
